# Pitstop Logo Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the 🏁 emoji placeholder in the drawer header with a flush 2×2 checker SVG mark (variant A1 from the visual brainstorm), and ship as v0.3.24.

**Architecture:** Inline SVG inside `<div class="mark">` in `Header.tsx`. The two amber rectangles render via `currentColor`, with the surrounding `.dheader .mark` rule supplying `color: var(--amber)`. Emoji-specific CSS (`font-size`, `line-height`) is removed since SVGs size themselves through `width`/`height` attributes. No new files, no new components — one usage doesn't justify a `Logo.tsx`.

**Tech Stack:** Solid.js (TSX), CSS (vanilla), Bun (build), Vite (inject bundle build), the existing `release.ts` script (test → bump → build → tag → push).
